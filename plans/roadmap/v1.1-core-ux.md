# v1.1 — Core UX Refinements

> Refining the v1.0 experience with better sorting and pagination fixes.

A minor milestone focused strictly on UX improvements to the core logbook before moving on to new major features, improving autocomplete relevance and resolving active bugs.

## Epics

- [UX Refinements](v1.1-core-ux/01-ux-refinements.md) — Autocomplete sorting, filter sorting, pagination fix.

## Goals

- Autocomplete fields should prioritize recently used values.
- Filter dropdown lists should sort options by their occurrence frequency.
- Paginated table should faithfully respect the "per page" count.
