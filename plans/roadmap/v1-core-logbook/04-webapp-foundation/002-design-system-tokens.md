# Design System & Tokens

Create the SkyBook design system with Tailwind theme tokens, typography, and color palette.

## Acceptance Criteria

- [ ] `webapp/src/style.css` with Tailwind `@theme` customization
- [ ] Color palette: deep navy/charcoal base, sunset orange → teal gradient accents
- [ ] Typography: Inter (body), JetBrains Mono (numbers/monospace)
- [ ] CSS custom properties for semantic colors (surface, text, accent, danger, warning)
- [ ] Dark mode as default (no light mode toggle in v1)
- [ ] Reusable utility classes for cards, buttons, inputs, badges
