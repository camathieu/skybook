# Responsive & Mobile

Make the jump list work well on mobile devices.

## Acceptance Criteria

- [ ] On screens < 768px: switch to card view (one card per jump)
- [ ] Card shows: #, Date, Dropzone, Type, key flags
- [ ] Swipe-to-delete gesture (with confirmation)
- [ ] Tap card to expand details / open edit
- [ ] Filter bar collapses to a "Filters" button on mobile
- [ ] Touch-friendly hit targets (min 44px)
